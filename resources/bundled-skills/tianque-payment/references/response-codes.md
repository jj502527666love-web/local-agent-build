# 天阙应答码

## 接口应答码 (code)

| code | msg | 说明 | 处理 |
|------|-----|------|------|
| 0000 | 操作成功 | 接口调用成功 | 检查 bizCode |
| 0001 | 系统异常 | 系统内部错误 | 重试 |
| 0002 | 报文格式错误 | JSON 格式问题 | 检查请求格式 |
| 0003 | 机构id异常 | orgId 不正确 | 检查 orgId |
| 0004 | 解密失败 | 数据解密失败 | 联系客服 |
| **0005** | 验签失败 | RSA 签名验证失败 | 见 [sign-examples.md](sign-examples.md) |
| 0006 | 该功能未开通 | 商户未开通此功能 | 联系客服开通 |
| 0008 | 未配置公钥 | 商户未上传公钥 | 上传公钥到天阙 |
| 0009 | 机构不存在 | orgId 不存在 | 检查 orgId |
| 0010 | 机构状态异常 | 机构被禁用 | 联系客服 |
| 0011 | 该机构已不合作 | 合作关系终止 | 联系客服 |

---

## 业务应答码 (bizCode)

### 成功

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 0000 | 成功 | 完成 |

### 支付中（需轮询）

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 2002 | 用户支付中，请稍后进行查询 | 轮询查询 |
| 2002 | 交易结果未知，请稍后查询 | 轮询查询 |

### 参数错误 (0002)

| bizMsg | 处理 |
|--------|------|
| 商编不能为空 | 传 mno |
| trmIp不能为空 | 传 trmIp |
| ordNo不能重复 | 换订单号 |
| ordNo,uuid至少有一个不能为空 | 传订单号 |
| origOrderNo,origUuid不能同时为空 | 退款需传原订单 |
| amt取值范围是[0.01-100000000] | 金额超限 |
| ordNo最大长度64 | 缩短订单号 |
| uuid最大长度32 | 缩短 uuid |
| 参数有误,请检查参数是否正确 | 检查参数 |

### 订单状态错误

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 2001 | 订单号重复，请使用唯一订单号 | 换订单号 |
| 2003 | 交易重复 | 换订单号 |
| 2005 | 找不到原交易单号 | 原订单不存在 |
| 0003 | 原交易未支付 | 等待支付完成 |
| 0004 | 原交易已退款 | 已退款 |
| 0005 | 交易已关闭 | 重新下单 |

### 退款错误

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 1002 | 留存资金不足以做此交易 | 余额不足 |
| 1003 | 退货金额超出可退金额 | 退款金额超限 |
| 0001 | 该时间段不允许退款 | 稍后重试 |
| 0001 | 退款超出期限 | 超过退款期限 |

### 商户/权限错误

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 0001 | 该商户不存在 | 检查 mno |
| 0001 | 该商户状态不可用 | 联系客服 |
| 3001 | 该商户无此交易权限 | 联系客服 |
| 5001 | 商户不存在或不在该机构下 | 检查商户归属 |
| 5002 | 未查询到该商户，请核实 | 检查 mno |

### 付款码错误（被扫）

| bizCode | bizMsg | 处理 |
|---------|--------|------|
| 5001 | 交易失败，付款码有误 | 检查 authCode |
| 5006 | 失败，授权码过期 | 刷新付款码 |
| 5007 | 失败，授权码错误 | 检查 authCode |
| 5008 | 失败，授权码校验错误 | 检查 authCode |

### 用户/渠道错误

|---------|--------|------|

| 5004 | 商户公众号 appid 和用户 openid 不匹配 | 检查配置 |

### 系统错误（重试）

|---------|--------|------|

| 4002 | 资源繁忙，请稍后再试 | 重试 |
| 9001 | 失败，请联系客服 | 联系客服 |
| 9001 | 失败，请重试 | 重试 |

## 交易状态 (tranSts)

### 支付

| tranSts | 说明 | 处理 |
|---------|------|------|
| SUCCESS | 成功 | 完成 |
| PAYING | 支付中 | 轮询 |
| FAIL | 失败 | 重试/提示 |
| CLOSED | 已关闭 | 重新下单 |
| CANCELED | 已撤销 | 重新下单 |

### 退款

| tranSts | 说明 | 处理 |
|---------|------|------|
| REFUNDSUC | 退款成功 | 完成 |
| REFUNDING | 退款中 | 轮询查询 |
| REFUNDFAIL | 退款失败 | 检查原因 |

---

## 响应结构

```json
{
  "code": "0000",
  "msg": "操作成功",
  "respData": {
    "bizCode": "0000",
    "bizMsg": "成功",
    "tranSts": "SUCCESS"
  }
}
```
